# Deep Learning CNN - CIFAR-10 Model Comparison

Computer vision coursework project comparing CNN architectures on CIFAR-10.

## Portfolio Summary

This project trains and evaluates ResNet-18 and AlexNet-style models for image classification, with emphasis on model comparison, training behaviour, and evaluation beyond a single metric.

Key work:

- Trained and compared **ResNet-18** and **AlexNet** on CIFAR-10.
- Used data augmentation, regularisation, and learning-rate scheduling with `ReduceLROnPlateau`.
- Analysed training/validation loss, gradients, confusion matrices, and class-level behaviour.
- Compared model size and performance tradeoffs.
- ResNet-18 achieved **93.31% test accuracy** with approximately **11.17M parameters**.
- AlexNet achieved **87.77% test accuracy** with approximately **35.86M parameters**.

## Skills Demonstrated

- Computer vision model training
- CNN architecture comparison
- PyTorch or TensorFlow/Keras-style deep learning workflow
- Data augmentation and regularisation
- Confusion matrix analysis

## Why This Matters

The project shows practical model evaluation: not just whether a model works, but how architecture choice, parameter count, training stability, and class-level errors affect the final result.
